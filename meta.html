Accessibility & Privacy

♿ Accessibility Statement

This website aims to be accessible to everyone. It includes a skip link, semantic HTML, keyboard navigation support, readable font sizes, and colour contrast that meets WCAG 2.1 AA standards. If you find anything that could be improved, please get in touch via the contact page.

🔒 Privacy Statement

This website does not use cookies, trackers, or any third-party analytics. No personal data is collected from visitors. The contact form is a visual placeholder — no data is sent or stored without your explicit action.

🛠️ Built With

This site was built from scratch using pure HTML, CSS and JavaScript — no frameworks, no libraries (except Leaflet.js for the map). Designed and written together with Claude AI, committed to GitHub at every step. Fonts by Google Fonts (Amiri, DM Sans, DM Mono).
